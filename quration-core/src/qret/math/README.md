\page qret_math math module

qret で使用する数学関連の関数・クラスを実装する。

## コードマップ

- `qret/math/boolean_formula.h` : 論理式
- `qret/math/clifford.h` : クリフォード操作
- `qret/math/float.h` : 浮動小数点の近似
- `qret/math/fraction.h` : 連分数
- `qret/math/integer.h` : 整数
- `qret/math/lcu_helper.h` : Linear Combination of Unitaries (LCU) に関連した関数
- `qret/math/pauli.h` : パウリ文字列
